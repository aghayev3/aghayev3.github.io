01 IDRIS — Identity Detection & Response Intelligence System CAPSTONE TEAM LEAD INSTRUCTOR COMMENDED
Team AURORA  ·  AzCC Technion Blue Team Cohort 7  ·  Role: Team Lead & Core Developer

Self-hosted ITDR (Identity Threat Detection & Response) product — the Cohort 7 capstone. As team lead I designed the architecture, coordinated the build across the team, and drove the project from threat emulation through auto-remediation. Earned strong commendation from programme instructors for its technical depth and operational completeness.

IDRIS emulates the full on-prem Active Directory identity kill-chain: Kerberoasting, AS-REP roasting, DCSync, pass-the-hash, Golden/Silver tickets — then validates in real time whether Wazuh detected each technique with evidence. Auto-remediation is fully guardrailed: approval gate, dry-run mode, backup/rollback, and full audit log.

03 Active Directory Automation Framework
PowerShell  ·  IaC  ·  Active Directory

Two-script PowerShell IaC framework for automated, repeatable provisioning of enterprise Active Directory environments with security built in from the start. Generate-ADConfig.ps1 produces a stateless JSON config (OUs, Groups, Users, GPOs, tiered admin accounts). Deploy-ADEnvironment.ps1 is an idempotent DC deployer with dry-run mode, GPO security baselines, privilege separation via "a-" prefixed admin accounts, and PowerShell Transcription audit logging.
